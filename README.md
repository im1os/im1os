# IM1 OS

**The Operating System for Powersports Dealers**

IM1 OS is a modern, cloud-native dealership management platform built specifically for the powersports industry.

## Vision

Provide dealerships with a single platform to manage every aspect of their business, including:

- Customers
- Vehicles
- Service
- Parts
- Inventory
- Suppliers
- Purchasing
- CRM
- Ecommerce
- Accounting
- Reporting
- AI

## Technology Stack

- Backend: .NET 10
- Frontend: React + TypeScript
- Database: PostgreSQL
- Cache: Redis
- Search: Meilisearch
- Infrastructure: Docker
- Hosting: Contabo VPS
- Source Control: GitHub

## Guiding Principles

- Multi-tenant from day one
- API-first architecture
- Security by design
- Modular architecture
- Cloud-native deployment
- Documentation before implementation
